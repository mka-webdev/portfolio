# Wisdom

**Wisdom** is a collaborative Unity/C# 2D game project developed by **Team Mmm Mmm Jeans** as part of university game development work.

The game was publicly released as a playable WebGL build on itch.io:

[Play Wisdom on itch.io](https://mmm-mmm-jeans.itch.io/wisdom)

This folder contains portfolio-safe evidence for the project. It does not include the Unity source project, source code, assets, or project files.

---

## Project Summary

Wisdom is a 2D platformer with puzzle, inventory, dialogue, exploration, and feedback-driven iteration. The project combined game design, Unity development, C# scripting, playtesting, documentation, and team-based delivery.

The project was completed by a five-person student team. Although formal roles existed, the project was highly collaborative, with team members contributing across programming, design, documentation, art, sound, production, dialogue, mechanics, testing, and refinement.

Wisdom achieved a final assessment result of **100% / High Distinction**.
