## Wisdom

**Wisdom** was developed by a five-person student team using Unity and C#. Although formal roles existed, the project was highly collaborative, with team members contributing across programming, design, testing, documentation, art, sound, production, dialogue, and visual development.

My contributions included:

* gameplay programming support
* health potion interaction work
* puzzle and interaction design contribution
* inventory and board interaction design contribution
* playtesting strategy
* feedback collection and analysis
* feedback-driven iteration
* technical design documentation
* review and refinement of mechanics and player experience

The project was publicly released on itch.io and achieved a final assessment result of **100% / High Distinction**.
