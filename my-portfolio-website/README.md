# Portfolio Website

This folder contains the source code for my personal portfolio website:

[mkawebdev.com](https://mkawebdev.com)

The website serves as the front-end presentation layer for the wider portfolio repository, bringing together software testing, accessibility, project management, game development, and software development evidence in a single location.

---

## Purpose

The site was developed to:

* present project evidence in a structured format
* showcase software testing and QA work
* document accessibility-related learning and projects
* demonstrate project management and delivery experience
* provide a central location for portfolio content and supporting documentation

---

## Technology Stack

* Node.js
* Express.js
* EJS templating
* Bootstrap 5
* Custom CSS
* Font Awesome

---

## Site Sections

### Home

Portfolio overview and featured projects.

### Projects

Software development project archive.

### Project Management

Project leadership, delivery, testing, and collaboration experience.

### Accessibility

Accessibility learning, credentials, and accessibility-focused project work.

### About

Professional background and current direction.

### Contact

Professional contact and profile links.

## Features

* Responsive design
* Keyboard-accessible navigation
* Skip-to-content link
* Semantic HTML structure
* Accessible colour palette
* Reusable EJS partials
* Structured project pages
* Mobile-friendly navigation
* External link announcements for screen reader users

## Development Notes

The website is intentionally content-focused rather than design-heavy. The emphasis is on presenting project evidence, testing artefacts, accessibility work, project management experience, and software development projects in a clear and maintainable format.

The site continues to evolve alongside the portfolio as new projects, testing examples, accessibility activities, and professional development work are added.
